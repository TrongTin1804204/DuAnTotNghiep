# Dự án Mobile App

## Giới thiệu
Đây là một ứng dụng di động được phát triển bằng React Native và Expo. Ứng dụng cung cấp các chức năng chính sau:

## Các chức năng chính

### 1. Quản lý tài khoản
- Đăng nhập/Đăng xuất
- Quản lý thông tin cá nhân
- Bảo mật tài khoản

### 2. Quản lý sản phẩm
- Xem danh sách sản phẩm
- Tìm kiếm sản phẩm
- Chi tiết sản phẩm
- Quản lý giỏ hàng

### 3. Quản lý đơn hàng
- Tạo đơn hàng mới
- Theo dõi trạng thái đơn hàng
- Lịch sử đơn hàng


### 5. Quản lý nhân viên
- Thông tin nhân viên
- Phân quyền truy cập
- Quản lý ca làm việc


## Công nghệ sử dụng
- React Native
- Expo
- TypeScript
- Redux (Quản lý state)
- React Navigation (Điều hướng)
- Firebase (Backend)

## Cài đặt và chạy
1. Clone dự án
2. Cài đặt dependencies: `npm install`
3. Chạy ứng dụng: `npm start`

## Cấu trúc thư mục
- `/assets`: Chứa các tài nguyên như hình ảnh, font chữ
- `/public`: Chứa các file tĩnh
- `/page`: Chứa các component và màn hình của ứng dụng
- `/node_modules`: Chứa các thư viện đã cài đặt
